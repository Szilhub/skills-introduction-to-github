#Hausübung Konfigurationsmanagement

## 1. Github vorstellen
Bei der erste Übung wird Githubs Grundfunktionen vorgestellt, wie Repositories, Branches oder Commits und Pull Requests.

### 5 wichtigste Takeaways
1. Repository erstellen
2. Commit 
3. Push
4. Branches nutzen
5. Pull Requests*


---
## 2. Markdown vorstellen
Bei der zweite Übung wird Markdown als Kommunikations- und Dokumentationstool vorgestellt.

### 5 wichtigste Takeaways
1. Überschriften mit `#`, `##`, `###` 
2. Bilder mit Links hinzufügen, wichtig, dass dabei die Text to Speech berüchsichtigt wird.
3. "Edit File" verwenden um die Datei zu Bearbeiten.
4. Checklisten erstellen [x] oder [].
5. Die drei "-" um eine Textbox vorzuheben.
